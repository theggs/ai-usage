    **Response mapping function `map_kimi_response(resp: &KimiUsageResponse) -> Vec<QuotaDimension>`:**
    - For `resp.usage`: parse limit/remaining as u64 from Option<String>. Compute `remaining_percent = if limit > 0 { Some(((remaining * 100) / limit) as u8) } else { None }`. Create QuotaDimension with label "Weekly", remaining_absolute as "{remaining} / {limit}", reset_hint from resetTime, status derived from percent (>50 "healthy", 20-50 "warning", <20 "exhausted", None "unknown"), progress_tone derived similarly ("success"/"warning"/"danger"/"muted").
    - For first entry in `resp.limits`: same pattern with label "5h Window". Parse detail.limit/remaining.
    - Return vec of all non-empty dimensions.

    **HTTP call function `call_kimi_api(token: &str, preferences: &UserPreferences) -> Result<KimiUsageResponse, SnapshotStatus>`:**
    - Use `crate::proxy::build_agent(preferences)`. If proxy resolution fails, return `Err(SnapshotStatus::ProxyInvalid)`.
    - Send GET to `https://api.kimi.com/coding/v1/usages` with headers:
      - `Authorization: Bearer {token}`
      - `Content-Type: application/json`
    - Match response:
      - Ok(resp) -> parse JSON body into KimiUsageResponse. If parse fails, return Err(TemporarilyUnavailable { detail: "response parse failed" }).
      - Err(ureq::Error::Status(401, _)) -> Err(AccessDenied)
      - Err(ureq::Error::Status(403, _)) -> Err(AccessDenied)
      - Err(ureq::Error::Status(429, _)) -> Err(RateLimited { retry_after_minutes: 5 })
      - Err(ureq::Error::Status(code, _)) -> Err(TemporarilyUnavailable { detail: format!("HTTP {code}") })
      - Err(ureq::Error::Transport(e)) -> Err(TemporarilyUnavailable { detail: e.to_string() })

    **Main function `pub fn load_snapshot(preferences: &UserPreferences) -> ServiceSnapshot`:**
    1. Call `resolve_kimi_token(preferences)`. If None, return NoCredentials snapshot.
    2. Call `call_kimi_api(&token, preferences)`. If Err(status), return snapshot with that status and empty dimensions.
    3. Map response to dimensions via `map_kimi_response`. If empty, return NoData snapshot.
    4. Return Fresh snapshot with dimensions, source "kimi-code".

    **Progress tone derivation helper `fn derive_tone(percent: Option<u8>) -> (&'static str, &'static str)`:**
    Returns (status, progress_tone) tuple:
    - Some(p) if p > 50 -> ("healthy", "success")
    - Some(p) if p > 20 -> ("warning", "warning")
    - Some(_) -> ("exhausted", "danger")
    - None -> ("unknown", "muted")
