   ```rust
   #[derive(Deserialize)]
   struct GlmEnvelope<T> { data: Option<T> }

   #[derive(Deserialize)]
   struct GlmQuotaResponse { limits: Option<Vec<GlmLimit>>, level: Option<String> }

   #[derive(Deserialize)]
   struct GlmLimit {
       #[serde(rename = "type")]
       limit_type: Option<String>,
       #[serde(deserialize_with = "deserialize_flexible_i64", default)]
       unit: Option<i64>,
       #[serde(deserialize_with = "deserialize_flexible_i64", default)]
       number: Option<i64>,
       #[serde(deserialize_with = "deserialize_flexible_f64", default)]
       percentage: Option<f64>,
       #[serde(rename = "currentValue", deserialize_with = "deserialize_flexible_i64", default)]
       current_value: Option<i64>,
       #[serde(deserialize_with = "deserialize_flexible_i64", default)]
       total: Option<i64>,
       #[serde(rename = "nextResetTime", deserialize_with = "deserialize_flexible_i64", default)]
       next_reset_time: Option<i64>,
   }
   ```

3. A pure `decode_glm_response(body: &str) -> Result<GlmQuotaResponse, String>` function that tries `GlmEnvelope` first, falls back to bare `GlmQuotaResponse`.

4. A pure `map_glm_response(resp: &GlmQuotaResponse) -> Vec<QuotaDimension>` function that:
   - Iterates `resp.limits`
   - Maps `(type, unit, number)` to label string (see RESEARCH.md)
   - **CRITICAL: Clamps `usage_pct` BEFORE inversion** (review concern #4):

     ```rust
     let usage_pct = limit.percentage.unwrap_or(0.0).clamp(0.0, 100.0);
     let remaining_pct = (100.0 - usage_pct).round() as u8;
     ```
   - Converts `next_reset_time` (ms) to ISO 8601: `chrono::DateTime::from_timestamp(ms / 1000, 0).map(|dt| dt.to_rfc3339())`
   - Computes `remaining_absolute` from `total - current_value` when both present
   - Sets `progress_tone` based on `remaining_pct`

   - If no token found, return None

   - On malformed JSON: return `TemporarilyUnavailable` with parse error detail (review concern #2: graceful degradation)
