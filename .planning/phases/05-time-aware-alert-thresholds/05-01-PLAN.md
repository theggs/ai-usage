Add `export const getQuotaHealthSignal = (dimension: QuotaDimension, nowMs?: number): QuotaHealthSignal => { ... }` with the exact mapping below:
- call `getQuotaBurnRateDisplay(dimension, nowMs)`
- if the result exists and `pace === "far-behind"`, return `{ level: "danger", source: "pace", pace: "far-behind", status: "exhausted", progressTone: "danger" }`
- if the result exists and `pace === "behind"`, return `{ level: "warning", source: "pace", pace: "behind", status: "warning", progressTone: "warning" }`
- if the result exists and `pace === "on-track"`, return `{ level: "normal", source: "pace", pace: "on-track", status: "healthy", progressTone: "success" }`
- if no valid burn-rate result exists and `remainingPercent` is `undefined`, return `{ level: "normal", source: "none", status: "unknown", progressTone: "muted" }`
- otherwise reuse the existing static threshold helpers so fallback behavior remains:
  - `> 50` => `{ level: "normal", source: "fallback", status: "healthy", progressTone: "success" }`
  - `20..50` => `{ level: "warning", source: "fallback", status: "warning", progressTone: "warning" }`
  - `< 20` => `{ level: "danger", source: "fallback", status: "exhausted", progressTone: "danger" }`

Add deterministic aggregate helpers in the same file:
