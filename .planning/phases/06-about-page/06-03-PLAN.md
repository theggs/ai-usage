## Key Implementation Details from UI-SPEC.md

### Hero Section Spec:
- App icon: 64x64px, sourced from `src-tauri/icons/128x128.png` (use the 128px asset at 64px display for Retina clarity)
- Copy the PNG into `src/assets/icons/app-icon.png` so Vite can resolve it as a static import
- App name: "AIUsage" at 20px/semibold, text-slate-900, centered
- Version: "v0.1.0" at 13px/regular, text-slate-500, centered, 4px below name
- Hero vertical padding: 48px top, 32px bottom
- All elements centered horizontally

### Key-Value Card Spec:
- Card container: `rounded-2xl border border-slate-200 bg-white shadow-[0_18px_40px_-28px_rgba(15,23,42,0.2)]` (matches SettingsView card)
- Rows separated by: `divide-y divide-slate-100` (matches SettingsView)
- Row padding: `px-5 py-3.5` (matches SettingsView `rowClassName`)
- Row layout: `flex items-center justify-between` -- label left, value right
- Label: text-[13px] font-semibold text-slate-700
- Value: text-[13px] text-slate-500

### Key-Value Rows (in order):
| Row | Label (zh-CN) | Label (en-US) | Value | Interaction |
|-----|---------------|---------------|-------|-------------|
| 1 | License | License | "Apache 2.0" | Static text |
| 2 | GitHub | GitHub | Repository URL | Clickable -- opens in browser via shell.open |
| 3 | Dependencies | Dependencies | "{count} packages / {audit}" | Static text |

### Copyleft Warning Visual:
When `copyleftCount > 0`:
- The dependency value text changes from text-slate-500 to text-amber-700
- A small inline badge appears: `rounded-full bg-amber-50 border border-amber-200 px-2 py-0.5 text-[11px] font-semibold text-amber-700`
- Badge text: "{count} copyleft"

When `copyleftCount === 0`:
- Show "All permissive" in text-slate-500 (no special treatment)
- Do NOT show "0 copyleft"

### i18n Keys to Add:
| Element | Key | zh-CN | en-US |
|---------|-----|-------|-------|
| About page title (header) | aboutTitle | "About" | "About" |
| License row label | aboutLicenseLabel | "License" | "License" |
| GitHub row label | aboutGitHubLabel | "GitHub" | "GitHub" |
| Dependencies row label | aboutDependenciesLabel | "Dependencies" | "Dependencies" |
| Dep audit: all permissive | aboutDepsAllPermissive | "{count} packages / All permissive" | "{count} packages / All permissive" |
| Dep audit: copyleft found | aboutDepsCopyleftFound | "{count} packages / {copyleft} copyleft" | "{count} packages / {copyleft} copyleft" |

### GitHub Link Behavior:
- Rendered as a `<button>` styled as a link (not an `<a>` tag) to avoid default browser navigation inside the Tauri webview
- On click: call `import { open } from "@tauri-apps/plugin-shell"; open(url)` to open in default browser
- Visual: text-blue-600, hover:text-blue-700, hover:underline, cursor-pointer
- Truncate the displayed URL if it exceeds the available width

### Version Retrieval:
- Use `import { getVersion } from "@tauri-apps/api/app";` 
- Call `getVersion()` in useEffect to get version string
- Display as "v{version}"

### Dependencies Data:
- Import audit data: `import auditData from "../../generated/license-audit.json"`
- Fallback if file doesn't exist (dev mode without running audit):
  ```typescript
  const audit = (auditData as { totalPackages?: number; copyleftCount?: number }) || { totalPackages: 0, copyleftCount: 0 };
  ```

<task type="auto" tdd="false">
  <name>Task 3: Create AboutView component</name>
  <files>src/app/about/AboutView.tsx</files>
  <action>
    Create the src/app/about/AboutView.tsx component:
    
    ```tsx
    import { useEffect, useState } from "react";
    import { getVersion } from "@tauri-apps/api/app";
    import { open } from "@tauri-apps/plugin-shell";
    import appIcon from "../../assets/icons/app-icon.png";
    import { getCopy } from "../shared/i18n";
    import { useAppState } from "../shared/appState";
    
    // Import license audit data with fallback for dev mode
    let auditData: { totalPackages: number; copyleftCount: number } = { totalPackages: 0, copyleftCount: 0 };
    try {
      auditData = await import("../../generated/license-audit.json");
    } catch {
      // Fallback when audit file doesn't exist (dev mode)
    }
    
    const GITHUB_URL = "https://github.com/user/ai-usage"; // Adjust to actual repo URL
    
    export const AboutView = () => {
      const { preferences } = useAppState();
      const [version, setVersion] = useState<string>("");
      const copy = getCopy(preferences?.language ?? "zh-CN");
      
      useEffect(() => {
        void (async () => {
          try {
            const v = await getVersion();
            setVersion(v);
          } catch {
            // Fallback: try to get from import.meta.env or package.json
            setVersion("0.1.0");
          }
        })();
      }, []);
      
      const handleOpenGitHub = () => {
        void open(GITHUB_URL);
      };
      
      const audit = auditData || { totalPackages: 0, copyleftCount: 0 };
      const dependencyText = audit.copyleftCount > 0
        ? copy.aboutDepsCopyleftFound
            .replace("{count}", String(audit.totalPackages))
            .replace("{copyleft}", String(audit.copyleftCount))
        : copy.aboutDepsAllPermissive.replace("{count}", String(audit.totalPackages));
      
      return (
        <section className="pb-4">
          {/* Hero Section */}
          <div className="flex flex-col items-center pt-12 pb-8">
            <img
              src={appIcon}
              alt="AIUsage"
              className="h-16 w-16 rounded-xl"
            />
            <h1 className="mt-6 text-xl font-semibold text-slate-900">
              AIUsage
            </h1>
            <span className="mt-1 text-[13px] text-slate-500">
              v{version}
            </span>
          </div>
          
          {/* Key-Value Card */}
          <div className="overflow-hidden rounded-2xl border border-slate-200 bg-white shadow-[0_18px_40px_-28px_rgba(15,23,42,0.2)]">
            <div className="divide-y divide-slate-100">
              {/* License Row */}
              <div className="flex items-center justify-between px-5 py-3.5">
                <span className="text-[13px] font-semibold text-slate-700">
                  {copy.aboutLicenseLabel}
                </span>
                <span className="text-[13px] text-slate-500">
                  Apache 2.0
                </span>
              </div>
              
              {/* GitHub Row */}
              <div className="flex items-center justify-between px-5 py-3.5">
                <span className="text-[13px] font-semibold text-slate-700">
                  {copy.aboutGitHubLabel}
                </span>
                <button
                  onClick={handleOpenGitHub}
                  className="max-w-[60%] truncate text-[13px] text-blue-600 transition-colors hover:text-blue-700 hover:underline cursor-pointer"
                  type="button"
                >
                  github.com/user/ai-usage
                </button>
              </div>
              
              {/* Dependencies Row */}
              <div className="flex items-center justify-between px-5 py-3.5">
                <span className="text-[13px] font-semibold text-slate-700">
                  {copy.aboutDependenciesLabel}
                </span>
                <div className="flex items-center gap-2">
                  <span className={`text-[13px] ${audit.copyleftCount > 0 ? "text-amber-700" : "text-slate-500"}`}>
                    {dependencyText}
                  </span>
                  {audit.copyleftCount > 0 && (
                    <span className="rounded-full bg-amber-50 border border-amber-200 px-2 py-0.5 text-[11px] font-semibold text-amber-700">
                      {audit.copyleftCount} copyleft
                    </span>
                  )}
                </div>
              </div>
            </div>
          </div>
        </section>
      );
    };
    ```
    
    Note: The dynamic import approach may need adjustment - either use a static import at top level with try/catch wrapper, or use a state variable to hold the audit data after async import.
    
    Better approach for the audit data import:
    ```tsx
    import rawAuditData from "../../generated/license-audit.json";
    
    const auditData = (rawAuditData as { totalPackages?: number; copyleftCount?: number }) || { totalPackages: 0, copyleftCount: 0 };
    ```
    
    But this will fail in dev if the file doesn't exist. Use a conditional approach:
    
    ```tsx
    let auditData = { totalPackages: 0, copyleftCount: 0 };
    try {
      // @ts-expect-error - File may not exist in dev
      auditData = (await import("../../generated/license-audit.json")).default || { totalPackages: 0, copyleftCount: 0 };
    } catch {
      // Dev mode fallback
    }
    ```
    
    Actually, for build-time generated files that should always exist in production, use a static import with a module declaration. Create src/generated/license-audit.d.ts:
    ```typescript
    declare module "../../generated/license-audit.json" {
      const data: { totalPackages: number; copyleftCount: number; generatedAt: string };
      export default data;
    }
    ```
    
    Then import statically. For dev mode, ensure the file exists with default values.
